mediacomp - biblioteca python

A biblioteca mediacomp é uma implementação do MediaComp em Python nativo.
Computação com Mídias, ou MediaComp, é uma abordagem contextualizada para introdução à progração com o tema de manipulação de mídias (imagens, sons, videos e textos). O MediaComp foi criado por pesquisadores do GeorgiaTech em Jython, juntamente com uma IDE própria para o ensino: JES.
